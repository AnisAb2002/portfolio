À propos de Anis

Étudiant en Licence 3 Informatique, parcours Informatiques des Systèmes Embarqués et Interactifs (ISEI) à l'Université Paris 8. Je suis passionné par le développement web et mobile, un domaine dans lequel je souhaite me spécialiser.

Rigoureux, engagé et fortement motivé, j'accorde une importance particulière à la qualité de mon travail et à l'amélioration continue de mes compétences.

Je suis conscient du rôle stratégique du web et du mobile dans notre société ; mon rêve est de contribuer à la création de solutions numériques innovantes, utiles et performantes.

Sur le plan linguistique, je dispose d'un niveau C2 en français, attestant d'une maîtrise avancée de la langue. j'ai également un niveau technique en anglais, et j'en suis aux premiers stades de l'apprentissage de l'allemand, que je poursuis avec motivation.

Mon intérêt pour les jeux vidéo m'a conduit à explorer le développement 3D et interactif, notamment à travers Unity, en suivant la formation officielle Unity Learn.

En parallèle, je cultive un fort attachement au sport et à la lecture, ces deux disciplines m'accompagnent dans ma démarche de développement personnel.

Projets

ProtX - Application mobile

ProtX est une application mobile (Android) de vente de compléments alimentaires pour athlètes, permettant aux utilisateurs de parcourir des produits sportifs, les ajouter au panier, passer des commandes et gérer leur profil. Le projet utilise Firebase Firestore comme base de données ainsi que Room pour la gestion locale du panier.

Protx - Application Web

Site web de vente de compléments alimentaires pour athlètes conçu dans le cadre d'un projet de fin d'étude pour ma licence Informatique en 2025. Codé en HTML, CSS, JavaScript et PHP. Le lien vers le dépot git du projet est la version finale et fonctionnelle, le deuxième lien sert uniquement à montrer la front-end du site.

Pecunia

Pecunia est une application web pour gérer le budget personnel, elle permet de suivre les dépenses et les revenus. Avec une architecture moderne de Next.js pour le frontend et Django REST Framework pour le backend et permet les opérations CRUD sur les transactions et avec une interface moderne et intuitif.

Libri - Application mobile

Libri est une application développé avec Flutter permettant de gérer la lecture. On peut ajouter les livres lus ou en wishlist, les noter, ajouter des descriptions, supprimer les livres et synchroniser les données avec Firebase en toute sécurité.

Loqui

Une application web de messagerie en HTML, CSS et PHP. Elle inclut l'inscription, la connexion, un chat général, des messages privés, l'envoi de fichiers et la gestion d'une base de données. Ce projet universitaire vise à concevoir une plateforme de communication simple en consultant une base de données MySQL avec PHP.

To Do - Application mobile

Projet qui a pour but l'apprentissage du développement mobile avec Kotlin et Android Studio. C'est une application To-Do List simple permettant d'ajouter, afficher et gérer des tâches. Réalisée pour explorer ViewBinding, RecyclerView et Room

Tache à faire - Application Web

Ce projet est une application web simple de gestion de tâches (to-di list), c'est pour apprendre les bases de la création d'interfaces web dynamiques avec HTML, CSS et JavaScript. Elle permet d'ajouter, cocher, modifier ou supprimer des tâches, tout en affichant la progression.

Chrono - Application Web

Ce projet est réalisé dans un cadre d'apprentissage, il met en pratique la manipulation du DOM, la gestion des événements et l'application de styles dynamiques. C'est un chronomètre interactif en JavaScript permettant de démarrer, arrêter et réinitialiser le temps tout en affichant les heures, minutes, secondes et millisecondes.

Pac Man - Application Web

Ce projet consiste à recréer le jeu classique Pac-Man en JavaScript afin de se familiariser avec la manipulation du canvas, la gestion des événements clavier et la logique de déplacement des personnages. Il permet d'apprendre à structurer un code de jeu simple tout en intégrant graphismes, collisions et comportements aléatoires.

Gestion de Stock - Logiciel

Logiciel de gestion de stock en Python avec Tkinter, MySQL, NumPy : ajout, suppression, mise à jour et recherche de produits, sauvegarde en base de données et export des données vers Excel. Interface intuitive pour la gestion et le suivi des stocks.

Suivi médical - Logiciel

Dans le cadre d'un projet universitaire, nous avons développé ce logiciel de bureau en Python, destiné à aider un médecin dans son cabinet à gérer le suivi des consultations médicales de ses patients. Ce logiciel a été conçu en utilisant la bibliothèque Tkinter et repose sur le modèle d'architecture MVC (Modèle - Vue - Contrôleur).

Iris - Modèle de classification avec Python

Dans le cadre d'un projet universitaire avec Python, j'ai importé un dataset Iris, pré-traité et analysé les données, visualisé avec des graphiques, implémenté l'algorithme de classification K-Nearest Neighbors (KNN) pour l'apprentissage et l'évalué le modèle.

DeepAge - Prédiction de l'âge

DeepAge est une application Python utilisant Tkinter et TensorFlow pour prédire l’âge en temps réel à partir de la caméra. Elle détecte les visages avec OpenCV, prétraite les images, puis estime l’âge grâce à un modèle CNN entraîné sur le dataset UTKFace.

Smart Canne - Application Web

Ce projet est une application web que j'ai développé (HTML et CSS) pour une startup que nous avons créée dans le cadre d'un projet universitaire.

Modélisation UML - Protx

Modélisation complète du système du site web Protx avec UML avant l'implémentation. Contient les diagrammes de classe, cas d'utilisation, contexte, activité, séquence, composants et déploiement pour décrire l'architecture et les interactions du système.
